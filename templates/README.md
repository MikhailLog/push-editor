# Папка шаблонов

Эта папка предназначена для хранения шаблонов пуш-уведомлений.

## Как использовать:

1. **Сохранение шаблона:**
   - Создайте пуш-уведомление в редакторе
   - Перейдите на вкладку "Шаблоны"
   - Введите имя шаблона
   - Нажмите "Сохранить"
   - Файл будет скачан в формате JSON
   - Поместите скачанный файл в эту папку `templates/`

2. **Загрузка шаблонов:**
   - Перейдите на вкладку "Шаблоны"
   - Нажмите кнопку "Загрузить шаблоны из папки"
   - Выберите папку `templates/` в диалоге
   - Все шаблоны из папки будут загружены

3. **Формат файла шаблона:**
   - Каждый шаблон - это JSON файл
   - Имя файла должно заканчиваться на `.json`
   - Внутри файла должны быть поля: `name`, `data`, `thumb`, `created`

## Примечание:

- Старые шаблоны из localStorage браузера будут автоматически доступны при первой загрузке
- Для работы с папкой templates требуется браузер с поддержкой File System Access API (Chrome, Edge, Opera)
- В других браузерах используйте кнопку "Импорт .json" для загрузки отдельных шаблонов
