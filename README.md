# Редактор пуш-уведомлений

Редактор для создания анимированных пуш-уведомлений с экспортом в WebM и MP4.

## Структура проекта

```
Push - редактор/
├── index.html          # Главный HTML файл
├── css/
│   └── styles.css     # Стили
├── js/
│   ├── app.js         # Основная логика приложения
│   ├── state.js       # Управление состоянием и шаблонами
│   └── utils.js       # Вспомогательные функции
├── templates/         # Папка для сохранения шаблонов
│   └── README.md      # Инструкция по работе с шаблонами
└── README.md          # Этот файл
```

## Запуск

⚠️ **Важно**: ES6 модули не работают при открытии файла напрямую через `file://` протокол из-за политики CORS браузера.

### Вариант 1: Локальный сервер (рекомендуется)

**Python:**
```bash
cd "Push - редактор"
python3 -m http.server 8000
```
Затем откройте в браузере: `http://localhost:8000/index.html`

**Node.js:**
```bash
npx http-server -p 8000
```

**VS Code:**
- Установите расширение "Live Server"
- Правый клик на `index.html` → "Open with Live Server"

### Вариант 2: Chrome с отключенной безопасностью (не рекомендуется)

```bash
# macOS
open -na Google\ Chrome --args --allow-file-access-from-files

# Windows
chrome.exe --allow-file-access-from-files
```

## Использование

1. Откройте `index.html` через локальный сервер
2. Настройте параметры в панели слева
3. Редактируйте текст двойным кликом (Enter - новая строка, Ctrl+Enter - сохранить)
4. Перемещайте элементы мышкой (плашка автоматически выравнивается по центру)
5. Экспортируйте в WebM или MP4 через вкладку "Экспорт"

## Возможности

- **Редактирование текста**: двойной клик для редактирования, многострочный текст (Enter)
- **Настройка стилей**: цвета, шрифты, размеры, выравнивание
- **Анимация**: настройка направления появления/ухода, длительности, эффекта нажатия
- **Шаблоны**: сохранение и загрузка шаблонов в папку `templates/`
- **Экспорт**: WebM и MP4 видео с настраиваемым FPS и битрейтом
- **Магнитное выравнивание**: плашка автоматически выравнивается по центру при перетаскивании
